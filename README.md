
# Predictive Maintenance Prediction System

## Overview
This project focuses on building a Predictive Maintenance Prediction System using Machine Learning techniques. The goal is to predict machine failures based on industrial sensor data such as air temperature, process temperature, rotational speed, torque, and tool wear.

The project includes:
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Outlier detection using boxplots
- Feature encoding and scaling
- Machine Learning model training
- Model evaluation using accuracy, confusion matrix, and classification report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset Features
The dataset contains:
- Air temperature [K]
- Process temperature [K]
- Rotational speed [rpm]
- Torque [Nm]
- Tool wear [min]
- Machine failure target column

## Machine Learning Models
The following models are used:
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

## Workflow
1. Import dataset
2. Perform data cleaning
3. Analyze dataset statistics
4. Visualize data using boxplots
5. Encode categorical values
6. Split dataset into training and testing sets
7. Scale numerical features
8. Train machine learning models
9. Evaluate model performance

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
